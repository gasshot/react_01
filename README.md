# React
리액트 연습

01. Node.js 설치하기 - https://nodejs.org/ko/download/

02. ctrl + shift + ` : 터미널 열기

03. npx create-react-app 본인이 원하는 폴더명

04. cd 본인이 원하는 폴더명

05. npm start


본인이 원하는 폴더명

├── node_modules
├── package.json
├── .gitignore
├── public

│   ├── index.html

│   ├── logo512.png
│   ├── manifest.json
│   └── robots.txt
└── src
    ├── App.css
    ├── App.js
    ├── App.test.js
    ├── index.css
    ├── index.js
    ├── logo.svg
    ├── serviceWorker.js
    └── setupTests.js
